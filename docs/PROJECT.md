# Korea Auto Market

## Project Type

Telegram Mini App

## Goal

Build a marketplace for buying and selling used cars in South Korea.

## Target Audience

- Uzbek users in Korea
- Korean users

## Core Features

- Telegram Login
- Create Listing
- Upload Images
- Search Cars
- Advanced Filters
- Favorites
- Seller Contact
- Admin Dashboard

## Messaging System

Users can contact sellers through the Telegram Bot.

No Telegram username required.

No internal realtime chat.

The bot acts as a message relay between buyers and sellers.

### Benefits

- Works without username
- Privacy friendly
- Simple MVP implementation
- Low infrastructure cost

## Infrastructure

| Layer | Technology |
|-------|------------|
| Frontend | Next.js |
| Database | PostgreSQL (Vultr VPS) |
| ORM | Drizzle |
| Storage | Cloudflare R2 |
| Reverse proxy | Nginx |
| Containers | Docker + Docker Compose |
| CI/CD | GitHub Actions |
| Deployment | Vultr VPS |

## Priority

1. Low Cost
2. Fast Performance
3. Mobile First
4. Easy To Scale

## Maximum MVP Budget

Under $20/month
