# Korea Auto Market

## Project Type

Telegram Mini App

## Goal

Build a marketplace for buying and selling used cars in South Korea.

## Target Audience

- Uzbek users in Korea
- Korean users

## Core Features

- Telegram Login
- Create Listing
- Upload Images
- Search Cars
- Advanced Filters
- Favorites
- Seller Contact
- Admin Dashboard

## Infrastructure

| Layer | Technology |
|-------|------------|
| Frontend | Next.js |
| Database | Neon PostgreSQL |
| ORM | Drizzle |
| Storage | Cloudflare R2 |
| Deployment | Vercel |

## Priority

1. Low Cost
2. Fast Performance
3. Mobile First
4. Easy To Scale

## Maximum MVP Budget

Under $20/month
